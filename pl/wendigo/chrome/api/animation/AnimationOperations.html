Package 

Class AnimationOperations

  • 
public final class AnimationOperations

    AnimationOperations represents Animation protocol domain request/response operations and events that can be captured.

    This API is marked as experimental in protocol definition and can change in the future.

    • Method Summary

      Modifier and Type Method Description
      final Single<ResponseFrame> disable() Disables animation domain notifications.
      final Single<ResponseFrame> enable() Enables animation domain notifications.
      final Single<GetCurrentTimeResponse> getCurrentTime(GetCurrentTimeRequest input) Returns the current time of the an animation.
      final Single<GetPlaybackRateResponse> getPlaybackRate() Gets the playback rate of the document timeline.
      final Single<ResponseFrame> releaseAnimations(ReleaseAnimationsRequest input) Releases a set of animations to no longer be manipulated.
      final Single<ResolveAnimationResponse> resolveAnimation(ResolveAnimationRequest input) Gets the remote object of the Animation.
      final Single<ResponseFrame> seekAnimations(SeekAnimationsRequest input) Seek a set of animations to a particular time within each animation.
      final Single<ResponseFrame> setPaused(SetPausedRequest input) Sets the paused state of a set of animations.
      final Single<ResponseFrame> setPlaybackRate(SetPlaybackRateRequest input) Sets the playback rate of the document timeline.
      final Single<ResponseFrame> setTiming(SetTimingRequest input) Sets the timing of an animation node.
      final Flowable<AnimationCanceledEvent> animationCanceled() Event for when an animation has been cancelled.
      final Flowable<AnimationCreatedEvent> animationCreated() Event for each animation that has been created.
      final Flowable<AnimationStartedEvent> animationStarted() Event for animation that has been started.
      final Flowable<Event> events() Returns flowable capturing all Animation domains events.
